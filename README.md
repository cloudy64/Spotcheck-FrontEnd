Description :
SpotCheck is a full-stack web application designed to help students find available study spaces across campus cafés in real-time. The platform eliminates the frustration of walking to a café only to find it completely full by providing live seat availability updates.


Technologies Used
1. JavaScript

2. HTML
Provides the structure and layout of the web pages.

3. CSS
Responsible for styling the application.

4. Express.js
Framework used to build the backend server and manage routes.
